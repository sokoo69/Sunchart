# ☀️ SunCart – Summer Essentials Store

## Project Name
**SunCart**

## Purpose
A modern summer eCommerce platform where users can explore and purchase seasonal products like sunglasses, summer outfits, skincare, beach accessories, and more. Users can browse products, view details, and place orders after authentication. The platform focuses on a premium user experience with responsive design and modern aesthetics.

## Key Features
*   **Authentication**: Secure login and registration using BetterAuth (Email/Password & Google OAuth).
*   **Protected Routes**: Product details and My Profile sections are fully secured and only accessible to logged-in users.
*   **User Profile Management**: Users can view their profile data and update their personal information (Name and Photo URL).
*   **Responsive Design**: Fully responsive layout perfectly optimized for mobile, tablet, and desktop devices.
*   **Interactive UI**: Smooth animations, hover effects, and a dynamic slider using Animate.css and modern design principles.
*   **Product Catalog**: View popular products, filter items by category, and read product details.

## NPM Packages Used
*   `next` - React framework (App Router)
*   `react` / `react-dom` - UI library
*   `better-auth` - Authentication management
*   `mongodb` - Database adapter for BetterAuth
*   `tailwindcss` - Utility-first CSS framework (v4)
*   `daisyui` - Component library for Tailwind CSS
*   `animate.css` - CSS animation library
*   `react-hot-toast` - Toast notifications
*   `react-icons` - Icon library
